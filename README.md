# 🖱️ AI Virtual Mouse

An AI-powered virtual mouse built using Python, OpenCV, MediaPipe, and PyAutoGUI.  
This project allows users to control the computer cursor using hand gestures through a webcam.

---

## 🚀 Features

- ✨ AI-powered virtual mouse control
- 🖱️ Cursor movement using index finger
- 👆 Left click using Thumb + Index gesture
- 👆👆 Double click using two-finger gesture
- 📷 Real-time hand tracking with webcam
- ⚡ Smooth and responsive movement
- 🤖 Built using Computer Vision and AI

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Computer Vision

---

## 📦 Installation

Install dependencies:

```bash
pip install opencv-python mediapipe pyautogui
```

Run the project:

```bash
python virtual_mouse.py
```
